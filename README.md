##Installation

Clone the repository

    git clone https://github.com/mamyhugues/test-ctrlweb.git


Switch to the frontend folder next

    cd ../frontend/

Install all the dependencies using npm

    npm install

serve with hot reload at localhost:3000

    yarn dev
